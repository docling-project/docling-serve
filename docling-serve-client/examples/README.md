# Client SDK Examples

Run these examples from the `docling-serve-client` directory:

```bash
uv run python examples/convert_compat.py
uv run python examples/task_api.py
uv run python examples/batch_and_chunk.py
```

The examples use the PDF fixtures that now live next to the client package tests:

- `tests/2206.01062v1.pdf`
- `tests/2408.09869v5.pdf`
